Projects

Neresde - Smart shopping assistant

Neresde is a smart shopping assistant that utilizes crowd-sourcing to display the latest product prices in markets all over Cyprus. All they have to do is scan their receipts!

Features:

  • - Product querying using keyword or barcode

  • - Recommended products specific for each user

  • - Price reporting

  • - Scanning receipts/labels

  • - Viewing submitted scans and their status

  • - Submitting request to add a market

  • - Moderators handling price reports

Technologies used:

  • - Flutter

  • - Laravel (PHP)

  • - MySQL

  • - Google Cloud Vision API

  • - HTML | CSS | Bootstrap

Development process:

The project involved a team of 4 developers, which I led. It took 8 months to complete, which included the following:

  • - User and system requirements analysis

  • - Functional and non-functional requirements analysis

  • - Use case analysis

  • - Architectural design

  • - Software development using SCRUM agile development

  • - Extensive testing via unit tests, integration tests, performance tests, user tests

My responsibilities

  • - Creation and maintenance of MySQL database

  • - Creation of required Laravel models and migrations

  • - Authentication services

  • - Creation and management of REST API to provide all services required

  • - Management of external API services such as Google cloud vision

  • - Creation and management of queued jobs to handle asynchronous background jobs such as receipt/label processing
